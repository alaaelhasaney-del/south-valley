# Running the Academy Management System

## Plan Steps:

1. [x] Created .env template - UPDATE DATABASE_URL with your Postgres connection string and run schema.sql
2. Install backend deps
3. Install frontend deps in dashboard-electron/
4. Start backend: npm run dev
5. Start Electron dashboard: cd dashboard-electron && npm run electron:dev

## Commands to run:

- npm install
- cd dashboard-electron && npm install
- npm run dev
- (new terminal) cd dashboard-electron && npm run electron:dev

Program runs with Electron app open!
