## 🚀 خطوات البدء السريع

### **1️⃣ إعداد البيئة (5 دقائق)**

```bash
# أضف إلى .env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-public-key
```

### **2️⃣ إنشاء Storage Bucket (2 دقائق)**

- اذهب إلى Supabase Dashboard
- Storage → Create Bucket
- الاسم: `images`
- اجعله Public

### **3️⃣ تثبيت المكتبات (1 دقيقة)**

```bash
npm install
```

### **4️⃣ تطبيق الترحيل (2 دقيقة)**

```bash
npm run db:migrate
```

### **5️⃣ دمج في server.js (10 دقائق)**

- انسخ كود من `READY_TO_USE_CODE.js`
- أضفه إلى server.js حسب التعليقات

### **6️⃣ اختبر النظام (5 دقائق)**

```bash
npm run test:images
```

### **7️⃣ ترحيل الصور القديمة (اختياري)**

```bash
npm run migrate:photos
```
