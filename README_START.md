## 🚀 البدء السريع

### 1. اقرأ وتجهز (5 دقائق)

```bash
# اقرأ هذا أولاً:
START_HERE.md
```

### 2. أضف متغيرات البيئة (2 دقيقة)

```env
SUPABASE_URL=your-url
SUPABASE_KEY=your-key
```

### 3. ثبت المكتبات (1 دقيقة)

```bash
npm install
```

### 4. طبّق الترحيل (5 دقائق)

```bash
npm run db:migrate
```

### 5. ادمج الكود (20 دقيقة)

انسخ من `READY_TO_USE_CODE.js`

### 6. اختبر (10 دقائق)

```bash
npm run test:images
```

**المجموع: 45 دقيقة ✅**

## 💻 API Endpoints

### رفع صورة

```bash
POST /api/students/:id/upload-image
```

### حذف صورة

```bash
DELETE /api/students/:id/image
```

### الحصول على الصورة

```bash
GET /api/students/:id/image
```
