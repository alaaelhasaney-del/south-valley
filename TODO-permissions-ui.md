## للاختبار:

```
cd dashboard-electron
npm install lodash  # إذا لم يكن مثبت
npm run dev
```

انتقل إلى `/roles` وجرب:

1. زر "تعديل" بجانب كل دور → يفتح modal
2. حدد/ألغِ checkboxes → اضغط حفظ
3. Hover على الصفوف
4. أضف رتبة جديدة
