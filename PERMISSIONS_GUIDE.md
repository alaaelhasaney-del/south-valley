# نظام إدارة الأكاديمية - دليل الاستخدام

## نظرة عامة

نظام شامل لإدارة الأكاديمية مع نظام صلاحيات متقدم يمكن لمدير النظام التحكم فيه بالكامل.

## الأدوار المتاحة

**قواعد تسمية الأدوار الجديدة:**

- حروف صغيرة (a-z) + أرقام (0-9) + شرطة سفلية (\_) فقط
- **يجب أن يبدأ بحرف** (لا يبدأ برقم أو \_)
- مثال صحيح: `new_role`, `manager_2024`
- مثال خاطئ: `NewRole`, `123role`, `_role`, `role-name`

### 1. مدير النظام (Admin)

- **الوصول**: كامل لجميع الفروع والموارد
- **الصلاحيات**: قراءة، إنشاء، تعديل، حذف لجميع الموارد
- **المهام**: إدارة المستخدمين، إدارة الصلاحيات، إدارة النظام بالكامل

### 2. مدير عام (General Manager)

- **الوصول**: عام (جميع الفروع)
- **الصلاحيات**: قراءة وكتابة محدودة لمعظم الموارد
- **المهام**: إدارة عامة للنظام مع قيود على المستخدمين

### 3. مدير فرع (Branch Manager)

- **الوصول**: فرعي (فرعه فقط)
- **الصلاحيات**: إدارة كاملة للطلاب والمالية في فرعه
- **المهام**: إدارة فرع واحد بالكامل

### 4. موظف (Employee)

- **الوصول**: فرعي (فرعه فقط)
- **الصلاحيات**: قراءة وكتابة محدودة
- **المهام**: مهام إدارية محدودة

### 5. معلم (Teacher)

- **الوصول**: فرعي (فرعه فقط)
- **الصلاحيات**: درجات الطلاب والحضور فقط
- **المهام**: إدارة الدرجات والحضور

### 6. محاسب (Accountant - محاسب)

- **الوصول**: كامل لجميع الفروع والموارد
- **الصلاحيات**: **كاملة (قراءة/إنشاء/تعديل/حذف)** لكل الموارد
- **المهام**: إدارة كاملة للمالية والحسابات والتقارير

## إدارة الصلاحيات

### لمدير النظام:

1. اذهب إلى صفحة "الأدوار" من القائمة الجانبية
2. اضغط على "تهيئة الصلاحيات الافتراضية" لإضافة جميع الصلاحيات
3. اختر الدور المراد تعديل صلاحياته
4. حدد الصلاحيات المطلوبة لكل مورد:
   - ✅ **قراءة**: يمكن قراءة البيانات
   - ✅ **إنشاء**: يمكن إضافة بيانات جديدة
   - ✅ **تعديل**: يمكن تعديل البيانات الموجودة
   - ✅ **حذف**: يمكن حذف البيانات
   - **نطاق الوصول**: عام (جميع الفروع) أو فرعي (فرعه فقط)

## الموارد المتاحة

- **المستخدمين**: إدارة حسابات المستخدمين
- **الطلاب**: بيانات الطلاب وإدارتهم
- **الفروع**: إدارة فروع الأكاديمية
- **الأقسام**: إدارة الأقسام الأكاديمية
- **المقررات**: إدارة المقررات الدراسية
- **المالية**: إدارة الحسابات والميزانيات
- **بنود المصروفات**: إدارة أنواع المصروفات
- **رسوم الطلاب**: إدارة الرسوم والمدفوعات
- **درجات الطلاب**: إدارة الدرجات والتقييمات
- **حضور الطلاب**: تسجيل الحضور والغياب
- **سجل النشاطات**: مراقبة أنشطة النظام
- **المخزن**: إدارة المخزون
- **التقارير**: إنشاء ومراجعة التقارير

## كيفية البدء

1. **تسجيل الدخول كمدير نظام**:
   - البريد الإلكتروني: admin@example.com
   - كلمة المرور: admin123

2. **تهيئة الصلاحيات**:
   - اذهب إلى صفحة الأدوار
   - اضغط "تهيئة الصلاحيات الافتراضية"

3. **تخصيص الصلاحيات**:
   - اختر الدور المطلوب
   - قم بتعديل الصلاحيات حسب الحاجة

## ملاحظات مهمة

- مدير النظام لديه صلاحية الوصول إلى كل شيء دائماً
- التغييرات في الصلاحيات تطبق فوراً
- يمكن تخصيص كل دور حسب متطلبات الأكاديمية
- النظام يدعم عدة فروع مع صلاحيات منفصلة

## الدعم الفني

لأي استفسارات أو مشاكل، يرجى مراجعة:

- ملفات السجل في مجلد logs
- إعدادات قاعدة البيانات في .env
- الكود المصدري في مجلد server.js
